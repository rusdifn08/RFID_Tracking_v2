Action Plan Prioritas

Prioritas dibuat berdasarkan dampak ke shipment delivery, akurasi RFID, dan kelancaran alur proses.

Prioritas WO / Area Masalah Tindakan PIC / Tim Terkait Tujuan
Catatan metodologi: Progress Shipment % dihitung dengan rumus Folding Out / Quantity Order × 100. Status akurasi RFID di sini adalah evaluasi flow check berbasis kesinambungan antar pos. Data rework/reject tidak tersedia pada file sumber, jadi gap antar proses dibaca sebagai potensi kombinasi antara WIP nyata, rework, dan/atau gap capture RFID. WIP snapshot dijadikan sumber utama quantity order dan posisi cumulative; file harian dipakai untuk PIC, aktivitas terakhir, dan rekonsiliasi data.
